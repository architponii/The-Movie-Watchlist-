# Movie Watchlist 🎬

## Project Description

This project is a **Movie Watchlist web application** that fetches movie data from a public API and displays it on a webpage.
The application shows movie posters, titles, and ratings dynamically.

The goal of this project is to practice **API integration, fetch requests, and responsive web design**.

---

## Features

* Fetch movie data from **TMDb API**
* Display movie information dynamically
* Show **loading message while data is fetching**
* Responsive design for **mobile, tablet, and desktop**

---

## Technologies Used

* HTML
* CSS
* JavaScript
* Fetch API
* TMDb API

---

## API Used

The project uses **The Movie Database (TMDb) API** to fetch movie data.

Example API endpoint:

```
https://api.themoviedb.org/3/movie/popular?api_key=YOUR_API_KEY
```
